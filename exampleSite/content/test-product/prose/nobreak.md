## No break shortcode `nb`

Using the `nb` shorcode should stop words breaking across lines. A common use case would be on things like product names, such as {{<nb>}}NGINX Plus (nobreak) {{</nb>}} vs NGINX Plus (with break).

Usage

``` markdown
{{</*nb*/>}}NGINX Plus{{</*/nb*/>}}
```
